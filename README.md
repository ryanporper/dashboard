# Statify - Full-Stack Dashboard App

## Check out the [Live site](https://statify-dashboard.onrender.com/) (may have a little bit of loading time on first visit)

## Built using: 
**Backend**
- Mongoose
- Express.js
- Node.js
#
**Frontend**
- React.js
- Redux Toolkit 
- Redux ToolkitQuery
- Material UI
- Material UI Datagrid
- Nivo
